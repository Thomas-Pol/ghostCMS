Hoe moet je Ghost CMS installen locaal.

Stap 1. voor dat je ghost kan instaleren moet je eerst nodeJS downloaden op de computer
je kan nodeJS downloaden via deze link: https://nodejs.org/en/download/package-manager

Stap 2. open de terminal als administrator

Stap 3. run de command "npm install ghost-cli@latest -g" (zonder de quotes)

Stap 4. maak een eigen map aan met de command mkdir my-first-local-ghost-installation (naam van de map mag je zelf bepalen)

Stap 5. dan wil je navigeren naar de map die je net heb gemaakt door de command cd my-first-local-ghost-installation/

Stap 6. eenmaal in de goeie route moet je deze command gebruiken ghost install local

Stap 7. en dan moet je een account aan maken en kan je aan de slag met ghost

tip: als je ghost weer wilt op starten nadat je computer is uit geweest. moet je weer via de terminal naar de goeie map
en run je de command ghost start
